Language Server Protocol

https://microsoft.github.io/language-server-protocol/ 

And they make cool things. Like defining something obvious
as language server protocol. 


it just enables different tools to speak different language
servers. you implement something once and it works everywhere.
you implement LSP in your editor, IDE or any tool once, and
can use all the servers out there.
you implement a LSP server once and it can be used in any tool
or IDE.
what the fuck were all those IDE developers thinking before?
