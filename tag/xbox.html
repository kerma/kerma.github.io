notes to self : How to remove unnecessary audio from a video file

Tue 30 July 2013

Once in a while I encounter a video file which has more than one audio stream attached to it. Usually I'm only interested in one of those and want to get rid of the others. Specially if the first one is not the one I'm interested in. Xbox does provide an easy way to selecting some other stream besides the first one.

So let's fire up avprobe first:

:> avprobe something_nice.avi

    Stream #0.1: Audio: ac3, 48000 Hz, 5.1, s16, 448 kb/s

I'm interested in the second audio stream, which is in the language I understand. avconv helps:

>:avconv -i something_nice.avi -map 0:0 -c:v copy -map 0:2 -c:a \
copy something_nice_eng.avi
